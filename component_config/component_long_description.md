The Sage Intacct Extractor enables you to extract data from your Sage Intacct cloud financial management system directly into Keboola Storage.

**Key Features:**
- **OAuth 2.0 Authentication**: Secure authentication with automatic refresh token rotation
- **Multiple Endpoints**: Configure multiple Sage Intacct objects (customers, invoices, vendors, accounts, etc.) in a single extraction
- **Incremental Loading**: Extract only new or modified records to optimize API usage and reduce processing time
- **Dynamic Schema Discovery**: Automatically discover available objects and fields from your Sage Intacct instance
- **Flexible Configuration**: Choose specific fields to extract or get all available data
- **Efficient Pagination**: Handle large datasets with cursor-based pagination

Perfect for building financial reporting dashboards, consolidating multi-entity data, or integrating Sage Intacct with other business systems.
